For the fastest access, add the site to your home screen or desktop so it opens like an app.

On iPhone or iPad

Open the link in Safari, tap Share, choose Add to Home Screen, turn on Open as Web App, then tap Add.

On Android

Open the link in Chrome, tap the 3-dot menu, then choose Add to Home screen or Install app if that option appears.

On desktop

Open the site in Chrome, click the 3-dot menu, then go to Save and share → Install Page as App.
